# Simple System Monitoring Script

Dieses Projekt ist ein kleines Lernprojekt zur Anzeige grundlegender
Systeminformationen.

## Funktionen
- Anzeige von Betriebssystem und Hostname
- CPU-Auslastung
- Arbeitsspeicher-Nutzung
- Festplatten-Auslastung

## Zweck
Das Skript wurde erstellt, um Grundlagen der Systemüberwachung und
des Skriptings kennenzulernen.

## Hinweis
Das Projekt dient ausschließlich zu Lernzwecken und wurde in einer
lokalen Testumgebung entwickelt.
